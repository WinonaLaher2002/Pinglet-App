# Security Policy

## Supported Versions

Currently, only the latest version of Pinglet is supported with security updates.

| Version | Supported |
|--------|-----------|
| 1.0.x  | ✅ |
| < 1.0  | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in Pinglet, please report it responsibly.

You can report issues by:

• Opening an issue on the GitHub repository  
• Contacting the developer directly

Please include the following details when reporting a vulnerability:

- Description of the issue
- Steps to reproduce the problem
- Screenshots or logs if available
- Your environment (OS, version)

Thank you for helping keep Pinglet safe and secure.
